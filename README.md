# 情系华一

## 开发

通过 PhotoShop 对官方发布的头像进行预处理，提取出文字图层。用 Illustrator 打开预处理后的 PSD 文件，进行图像描摹后导出为 SVG 格式。删除 SVG 中可能多余的路径。最后使用 HTML 的 Canvas 进行头像生成。
